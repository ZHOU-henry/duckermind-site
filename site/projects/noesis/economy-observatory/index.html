Method Layer

How the prototype should be read

Measured where possible

Use current public country indicators

Compute, talent, publications, GitHub, and electricity layers now tie back to explicit public data sources.

Carry-forward where needed

Do not invent annual movement

When no newer country-level release exists, the observatory now carries forward the latest measured value instead of fabricating 2025 or 2026 jumps.

Composite by design

Keep proxy quality explicit

Some factors are directly measured more cleanly than others, so the interface now separates stronger measured layers from weaker proxy layers.
