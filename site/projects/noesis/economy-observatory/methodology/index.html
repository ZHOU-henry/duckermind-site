Methodology

Prefer measured inputs and explicit proxy labels.

This observatory build removes the earlier synthetic annual bump logic. Where newer country-level releases are unavailable, the system now carries forward the latest measured public values instead of guessing.

Data Rule

How the year slices should be interpreted

Slice Meaning Why this is safer
2024 Published country snapshot using measured public-source inputs. Keeps the baseline tied to real released indicators.
2025 Carry-forward view where country-level updates remain incomplete. Avoids synthetic interpolation and false annual motion.
2026 Current observatory synthesis using the latest published country values plus refreshed cloud-footprint inventory. Lets the interface stay current without pretending hidden data exists.

What changed

The old 2025 / 2026 synthetic bump is gone

The earlier prototype mechanically inflated values by year. That logic has been removed because it created motion without new evidence.

Why China now ranks higher on energy

Scale matters more than clean-share optics alone

The current energy factor weights absolute electricity and low-carbon electricity capacity heavily, which is closer to AI-economy power readiness than a climate-only lens.

Known limitation

Not every factor has a perfect country-level public dataset

The data factor remains explicitly proxy-labeled because open, comparable country-level dataset-access metrics are still weak.
